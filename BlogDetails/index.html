6 Basic methods of the JavaScript array that every JS developer should know - reactJS series(ep-5)

1) Array.toString() >> It returns an array as a comma-separated string. let foods= [‘Cheese’, ‘Egg’, ‘Butter’, ‘Margarine’, ‘Yogurt’, ‘Cream’]; console.log(foods.toString()) // Cheese,Egg,Butter,Margarine,Yogurt,Cream

2) Array.join() >> This method joins array elements into a string. let foods= [‘Cheese’, ‘Egg’, ‘Butter’, ‘Margarine’, ‘Yogurt’, ‘Cream’]; console.log(foods.join(‘+’)) // Cheese+Egg+Butter+Margarine+Yogurt+Cream

3) Array.push() >> This method adds a new element to an array. let foods= [‘Cheese’, ‘Egg’, ‘Butter’, ‘Margarine’, ‘Yogurt’, ‘Cream’]; foods.push(‘Meat’) console.log(foods) // [“Cheese”, “Egg”, “Butter”, “Margarine”, “Yogurt”, “Cream”, “Meat”]

4) Array.pop() >> This method removes an element (last element) to an array. let foods= [‘Cheese’, ‘Egg’, ‘Butter’, ‘Margarine’, ‘Yogurt’, ‘Cream’]; foods.pop() console.log(foods) // [“Cheese”, “Egg”, “Butter”, “Margarine”, “Yogurt”]

5) Array.shift() >> This method removes an element (firstelement) to an array. let foods= [‘Cheese’, ‘Egg’, ‘Butter’, ‘Margarine’, ‘Yogurt’, ‘Cream’]; foods.shift() console.log(foods) // [“Egg”, “Butter”, “Margarine”, “Yogurt”, “Cream”]

6) Array.unshift() >> This method adds a new element to the beginning of an array let foods= [‘Cheese’, ‘Egg’, ‘Butter’, ‘Margarine’, ‘Yogurt’, ‘Cream’]; foods.unshift(‘Meat’) console.log(foods) // [“Meat”, “Cheese”, “Egg”, “Butter”, “Margarine”, “Yogurt”, “Cream”]
